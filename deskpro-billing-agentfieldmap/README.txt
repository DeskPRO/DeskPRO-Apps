This simple app that sets a default value on a custom 'choice' billing field. You set a map of `agentId=valueId` you want to set, and when an agent views the billing form, the value for the field is the value in your map.

## Packaging this app

To package this app, just create a ZIP of this directory.

## Installing this app

Once you have created a ZIP of this directory, you can install the app from Admin > Apps > Upload App
